Início HTML CSS

O que é CSS?

Introdução ao CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos HTML. Ele permite separar o conteúdo da apresentação, facilitando a manutenção e o design de páginas web.

Conceitos Básicos

Seletores

Seletores são usados para selecionar elementos HTML que você deseja estilizar. Exemplos de seletores básicos:

  • Seletores de tag
  • Seletores de classe
  • Seletores de ID

Propriedades e Valores

No CSS, você define o estilo de um elemento utilizando propriedades e valores. Por exemplo:

color: red;

Classes e IDs

Classes e IDs são utilizados para aplicar estilos específicos a elementos. Classes são definidas com um ponto (.) e IDs com um hashtag (#).

Cascata e Especificidade

CSS aplica estilos com base na cascata e especificidade. A cascata determina a ordem em que os estilos são aplicados, enquanto a especificidade define a prioridade dos seletores.
