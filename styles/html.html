O que é HTML?

Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem padrão utilizada para criar e estruturar páginas web. Ele utiliza tags para definir elementos como texto, imagens, links, listas e tabelas, permitindo que os navegadores exibam o conteúdo de forma organizada e estilizada.

Tags HTML Básicas

Parágrafos

A tag <p> é usada para criar parágrafos de texto.

Este é um parágrafo.

Listas

As listas podem ser ordenadas (<ol>) ou não ordenadas (<ul>), e os itens de lista são definidos com a tag <li>.

Tabelas

Tabelas são criadas com a tag <table>. Linhas de tabelas são definidas com <tr>, células de dados com <td>, e cabeçalhos de tabela com <th>.

Coluna 1 Coluna 2
Dados 1 Dados 2

Imagens

A tag <img> é usada para adicionar imagens a uma página. Deve-se incluir os atributos src e alt para definir o caminho da imagem e a descrição da mesma.

Links

A tag <a> é usada para criar links. O atributo href define o destino do link.
